文件结构

文件结构好像这个样子

JingYes/
├── less/
│   └── jingyes.less
├── doc/
│   └── index.html
├── demo/
│   └── index.html
├── jingyes.css
├── jingyes.min.css
├── jingyes.js
├── jingyes.min.js
└── img/
    ├── glyphicons-halflings.png
    └── glyphicons-halflings-white.png

less是源代码目录，doc就是您看到的这个文档目录，demo是演示例子目录，生产环境这些目录不必部署。

基本的HTML模版

一个典型的HTML文件

<!doctype html>
<html lang="zh_CN">
  <head>
    <meta charset="UTF-8">
    <title>JingYes CSS FrameWork</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="pathto/jingyes.css" rel="stylesheet">
    <link href="pathto/yourdoc.css" rel="stylesheet">
  </head>
  <body>
    <h1>Hello Word!</h1>
  </body>
  <script src="pathto/jquery.js"></script>
  <script src="pathto/jingyes.js"></script>
</html>
