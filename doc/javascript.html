手风琴

给导航加一个class accordion 就变成了手风琴效果，标记是 nav dl.accordion ，当然还要有 dd dl嵌套。左侧的子导航已经加上了这个效果，点击子导航中的“手风琴”就可以看到效果。

<nav>
  <dl class="accordion">
    <dt><a href="#accordion">手风琴</a></dt>
    <dd>
      <dl>
        <dt>演示条目</dt>
        <dt>演示条目</dt>
      </dl>
    </dd>
  </dl>
</nav>

展开、折叠效果写在 jingyes.css 中。如果您觉得使用 nav 有所不妥，可以用 div.nav dl 这种形式替代 nav dl。后续类似情况同理
